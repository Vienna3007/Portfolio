## ⚙️ Run Locally

Clone the repository

```bash
git clone https://github.com/Vienna3007/Portfolio.git
```

Navigate into project directory

```bash
cd Portfolio/portfolio-new
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production version

```bash
npm run build
```
